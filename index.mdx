Welcome to PDFShift's reference API.

PDFShift is an HTML to * conversion API. We currently support converting an HTML document toward the following types :

 * PDF - Read more at [Convert to PDF](/api-reference/convert-to-pdf)
 * PNG - Read more at [Convert to PNG](/api-reference/convert-to-png)
 * JPEG - Read more at [Convert to JPEG](/api-reference/convert-to-jpeg)
 * WEBP - Read more at [Convert to WEBP](/api-reference/convert-to-webp)


But you can also store your HTML data on our service for faster processing, by taking advances of our Template endpoint ([read more at "Templates"](/api-reference/templates/list-templates)).

The documents generated from our service can be directly stored on your own bucket.
We currently support **Amazon S3** and **Google Cloud Storage** and plan to support more in the future.
You can read more at [Export to your bucket](/docs/saving/to-s3)

Finally, we highly recommand you to read more about [how to run an authenticated request using your API Key](/docs/authentication),
and the possibility to run as many tests as you want for free using the `sandbox` parameter at [Sandbox mode](/docs/sandbox).

Feel free to read this documentation. If you find any mistake, or an error on how the API is explained, please let us know and we'll quickly fix it.

We remain available if you need any help, simply contact our support@ and we'll be happy to help :)
