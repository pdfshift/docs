---
title: "Getting an API Key"
mode: "center"
---

In order to generate PDF without the PDFShift's watermark, you will need an API Key.
You can request one by creating an account on <a href="https://pdfshift.io" target="_blank">PDFShift's website</a>.

Once you have submitted the form, we will send you an email containing your API Key.
You can also create other API Key if you want, to split your keys accross your project.
This can be done via <a href="https://app.pdfshift.io" target="_blank">your Dashboard</a>.

You can test and play with our API without an API Key.
Simply don't provide the `X-API-Key` header and the request will be done, with the difference that a watermark will be applied to the document.

<Note>Applying a watermark makes the generation slower, so you will have better result when using your API Key</Note>
